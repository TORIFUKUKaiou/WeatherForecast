# WeatherForecast

技術評論社刊「WEB+DB PRESS  Vol.102」の特集「ReactNative」のサンプルコードです。

## npm module のインストール

`$ npm install`

## 実行

#### iOS

```
$ react-native run-ios
```

自動で iOS のエミュレータが起動し、アプリが起動します。

#### Android で実行

Android のエミュレータを起動、ないし実機を PC に接続した状態で、

```
$ react-native run-android
```

を実行すると、アプリが立ち上がります。
